# Maintainers

This file documents the current maintainership model for Claude Local Stack.

## Current Maintainer

- Repository owner: [`@code-sport`](https://github.com/code-sport)

## Maintainer Responsibilities

Maintainers are expected to:

- review and triage issues
- review pull requests
- keep documentation and project metadata current
- manage releases and changelog updates
- coordinate security handling described in [`SECURITY.md`](SECURITY.md)
- enforce the community standards in [`CODE_OF_CONDUCT.md`](CODE_OF_CONDUCT.md)

## Decision Making

For now, project decisions are maintainer-led.

Community input through issues and pull requests is encouraged, but maintainers make the final merge and release decisions.

## Availability

This repository is maintained on a best-effort basis. Response times may vary depending on maintainer capacity.

## Becoming a Maintainer

There is no formal maintainer nomination process yet.

Contributors who consistently provide high-quality, trusted contributions may be invited by the current maintainer to take on a larger role.
